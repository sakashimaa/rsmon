## **rsmon**

rsmon is a activity monitor that tracks system resources usage (working only in Linux because of system-specific /proc directory)
It is build using Ratatui for terminal GUI, tokio runtime for async operations, native rust library to read / parse file contents

**Installation**

```bash
curl -fsSL https://raw.githubusercontent.com/sakashimaa/rsmon/main/install.sh | sh
```
